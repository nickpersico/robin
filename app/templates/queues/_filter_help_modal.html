How to copy filter JSON from Close

{# Video #} {# Steps #}
  1. Navigate to Leads in Close.
  2. Add the filter conditions you want Robin to look for. We recommend including a condition where the User Custom Field you're using to assign leads is set to "not present" — so only unassigned leads get picked up.
  3. Once your filters are set, click the three-dots (⋯) icon in the top-right menu and select "Copy Filters (JSON)".
  4. Paste the copied JSON into the Lead filter field in Robin.
